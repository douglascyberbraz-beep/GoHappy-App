# KIDOA NEXT - Versión Profesional 🚀

Bienvenido a la versión profesional de **Kidoa**, migrada a **Next.js** para ofrecer una experiencia premium, escalable y 100% funcional.

## Tecnologías Utilizadas
- **Framework**: Next.js 14 (App Router)
- **UI & Animaciones**: Tailwind CSS + Framer Motion
- **Backend**: Firebase (Auth & Firestore)
- **IA**: Google Gemini 1.5 Flash
- **Mapas**: MapLibre GL 3D

## Estructura del Proyecto
- `/src/app`: Rutas y lógica de la aplicación.
- `/src/components`: Componentes premium reutilizables (Nav, Mapa, etc.)
- `/src/services`: Capa de servicios para Firebase e Inteligencia Artificial.
- `/public`: Assets, logos e iconos.

## Cómo Ejecutar (Desarrollo)
1. Abre una terminal en esta carpeta.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

> Si no tienes Node.js en tu PATH, he localizado uno en tu sistema que puedes usar:
